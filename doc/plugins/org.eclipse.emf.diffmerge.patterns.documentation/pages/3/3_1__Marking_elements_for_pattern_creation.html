Marking Elements for Pattern Creation
Before clicking “Create pattern…”, the user needs to select model elements. It is often convenient to operate in a diagram for that purpose; however, a diagram may not show all the elements needed. Conversely, using the Project Explorer allows the user to select all the elements he wants, but this operation can be cumbersome.

As a solution, it is possible to use the following menu items.

  • “Include in patterns”: mark the elements as selected (in a diagram or in the Project Explorer) for pattern creation. When the Pattern Creation wizard is being opened, these elements will appear together with those that are actually selected.


  • “Reset inclusions”: cancel the effect of the menu item above. Only the elements which are actually selected will appear in the Pattern Creation wizard.
